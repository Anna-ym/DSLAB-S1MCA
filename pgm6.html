Basic Properties

color: Sets the color of the text. p { color: blue; }
background-color: Sets the background color of an element. div { background-color: yellow; }
font-size: Sets the size of the font. h1 { font-size: 24px; }

Box Model Properties

margin: Sets the space around elements. .box { margin: 20px; }
padding: Sets the space between the content and the border of an element. .box { padding: 10px; }
border: Sets the border around an element. .box { border: 1px solid black; }

Layout Properties

display: Specifies how an element is displayed. .container { display: flex; }
position: Specifies the positioning method for an element. .element { position: absolute; top: 50px; left: 100px; }
float: Specifies whether an element should float to the left or right. .image { float: left; }
